# Como criar pacote de memória

## Quando usar

Ao encerrar conversa ou etapa com contexto relevante.

## Passo a passo

1. Usar `90_TEMPLATES/template_pacote_memoria.md`.
2. Registrar origem, projeto, data, tipo e status.
3. Extrair decisões, erros, conceitos, sugestões e próximos passos.
4. Sugerir arquivos de destino.
5. Classificar antes de consolidar.
