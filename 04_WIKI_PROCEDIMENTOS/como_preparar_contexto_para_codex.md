# Como preparar contexto para Codex

## Passo a passo

1. Identificar o projeto.
2. Ler contexto geral, decisões, arquitetura, erros e próximos passos.
3. Delimitar escopo e restrições.
4. Informar arquivos que podem ser alterados.
5. Definir critério de sucesso.

## Resultado esperado

Execução coerente com decisões já registradas.
